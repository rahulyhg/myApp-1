Sunrise and Sunset

This web app will let the user know the exact time in HH:MM:SS AM/PM format for sunrise and sunset of a particular location based on the search criteria.

The plan is to use Google's Places Autocomplete API to fetch Geo coordinates, latitude and longitude, when user searches in the input box below; along with autocompleting the place name.

Then the Sunrise Sunset API will be used to fetch the correct time of sunrise/sunset and display it to the user.
