Sunrise and Sunset

This webapp will let the user know the exact local time, in 24-Hour format, when sunrise and sunset will occur at the user searched location.

The plan is to use Google's Places Autocomplete API to autocomplete the place name and also fetching Geo Coordinates, latitude and longitude, when user starts typing in the input search box below.

Then using the Free Code Camp Weather API will be used to fetch the sunrise and sunset time. The results are then converted from UTC to localtime adjusted with timezone and daylight savings to display correct local time of sunrise and sunset using Google's Timezone API.
